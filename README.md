# 🎯 Image Carousel PRO

Modern and responsive image carousel built with **HTML, CSS, and vanilla JavaScript**.  
Designed with a clean dark UI and smooth user experience.

## 🚀 Features

- 🖼️ Large main image display
- 📝 Image title and caption
- ⬅️ Previous / Next navigation
- 🔢 Slide counter (e.g. 1 / 4)
- 🧭 Thumbnail navigation
- 🎯 Active thumbnail highlight
- 🔄 Autoplay functionality
- 🌫️ Smooth fade transition between slides
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (modern UI, dark theme, responsive design)
- Vanilla JavaScript (no frameworks)

---

## ⚙️ How It Works

The carousel is built using a simple and scalable logic:

- `slides` array stores all image data (image, title, caption)
- `currentIndex` keeps track of the current slide
- `updateSlide()` updates the UI dynamically
- Navigation buttons modify the index
- Thumbnails allow direct slide selection
- Autoplay cycles through slides automatically
